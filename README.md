# Email Alert Assistant

## Overview
Email Alert Assistant is a smart system that automatically detects important emails such as
offer letters, interview calls, and meeting invites using a Retrieval-Augmented Generation (RAG) approach.

## Technologies Used
- Python
- Transformers (FLAN-T5)
- SentenceTransformers
- ChromaDB
- Gmail API
- Streamlit

## Features
- Identifies critical emails automatically
- Uses RAG with keyword fallback for reliability
- Stores embeddings in ChromaDB
- Displays alerts through a Streamlit dashboard
